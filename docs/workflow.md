# Antigravity Workflow

## Autonomous pipeline
1. Auto-discover competitor channels in the target niche
2. Pull recent top-performing videos
3. Extract patterns from title, hook, thumbnail, pacing, retention, and upload time
4. Score niches by monetization, competition, and repeatability
5. Identify the best upload window for the niche and audience
6. Generate original video ideas
7. Rank ideas by CTR, retention, and monetization
8. Write cinematic scripts and scene plans
9. Create voice, visuals, subtitles, and thumbnails
10. Export long-form + short-form cinematic versions
11. Publish to platforms
12. Collect analytics
13. Update niche scores and upload-time scores
14. Improve the next batch automatically

## Platform rules
- YouTube long-form: 8–15 min, max 20 min
- Shorts / TikTok / Reels: 20–40 sec, max 60 sec
- Hook must happen in the first 3 seconds
- Pattern interrupt every 5–7 seconds

## Automation stack
- Python for orchestration and local processing
- n8n for workflow automation and scheduling
- YouTube API for uploads and analytics
- Optional connectors/APIs for TTS, video rendering, thumbnails, and subtitle generation
