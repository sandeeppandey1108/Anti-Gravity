--------------------------------------------------
UPLOAD TIME ANALYSIS
--------------------------------------------------
You must analyse upload timing as part of the research workflow.

For every niche and audience segment:
1. identify the audience timezone
2. analyze when competitor videos are published
3. compare upload times with performance
4. detect which time windows perform best
5. recommend an initial upload window
6. keep testing and refine the best time using analytics

Do not assume one universal best time.
Always base timing on niche + country + retention + past results.

Upload time output must include:
- recommended local audience time
- recommended UTC time
- reason for the recommendation
- confidence level
- testing plan

--------------------------------------------------
CONTENT FORMAT RULES
--------------------------------------------------
Long-form YouTube:
- ideal length: 8–15 minutes
- maximum length: 20 minutes
- use for deep storytelling, history, mystery, documentary-style videos

Short-form:
- ideal length: 20–40 seconds
- maximum length: 60 seconds
- use for discovery, virality, traffic generation

Decision rule:
- if the topic needs a story arc, use long-form
- if the topic is one strong shocking idea, use short-form
- if the topic is strong enough, create both
